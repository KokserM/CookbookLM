# Privacy Policy

CookbookLM is a Chrome extension that extracts recipe content from pages you choose and can send that recipe content to OpenAI using your own API key.

## Data processed

When you click `Generate PDF` or otherwise run AI processing, CookbookLM may send the recipe title, ingredients, preparation steps, source URL, selected output settings, and selected dietary restrictions to OpenAI. This is necessary to translate the recipe and reason about substitutions.

CookbookLM does not operate its own backend server and does not sell or share your data with any party other than the OpenAI API endpoint needed for the feature you request.

## API key storage

Your OpenAI API key is stored locally in Chrome extension storage on your device. CookbookLM does not include an OpenAI API key and does not transmit your key anywhere except directly to OpenAI as the authorization header for your request.

## Local cache

CookbookLM may cache AI recipe outputs locally in Chrome storage to avoid repeated API calls for the same recipe/settings combination.

## User control

You can remove your API key from the extension settings at any time. You can also remove the extension from Chrome to delete extension-managed local data.
